# GSD: New Milestone

Start a new version/milestone after completing the current one. Preserves history and initializes fresh roadmap for v2, v3, etc.

## Workflow Source

This workflow is defined in detail at `_gsd/workflows/new-milestone.md`.

**CRITICAL:** Load the FULL content from `@_gsd/workflows/new-milestone.md`, READ its entire contents, and follow its directions exactly!

## Prerequisites

Run `/gsd:complete-milestone` first.

## Usage

```
/gsd-new-milestone
```
