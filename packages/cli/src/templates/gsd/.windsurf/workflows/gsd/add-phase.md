# GSD: Add Phase

Append new phases to the end of an existing roadmap when scope expands or new requirements emerge.

## Workflow Source

This workflow is defined in detail at `_gsd/workflows/add-phase.md`.

**CRITICAL:** Load the FULL content from `@_gsd/workflows/add-phase.md`, READ its entire contents, and follow its directions exactly!

## Quick Reference

This command will:
1. Analyze current roadmap state
2. Gather requirements for new phase
3. Design phase specification with tasks
4. Append to end of ROADMAP.md as Phase {N+1}
5. Update PROJECT.md and STATE.md if needed

## Usage

```
/gsd-add-phase
```

Use this to extend the roadmap when new work is identified.
