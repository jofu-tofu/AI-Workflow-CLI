# GSD: Pause Work

Create comprehensive handoff documentation when suspending work for seamless resumption later.

## Workflow Source

This workflow is defined in detail at `_gsd/workflows/pause-work.md`.

**CRITICAL:** Load the FULL content from `@_gsd/workflows/pause-work.md`, READ its entire contents, and follow its directions exactly!

## Quick Reference

This command will:
1. Capture current work state and context
2. Document what you were doing and next steps
3. Identify blockers and mental model
4. Create HANDOFF.md with resumption checklist
5. Safely commit/stash uncommitted changes

## Usage

```
/gsd-pause-work
```

Use before context-switching, taking a break, or handing off to another person.
