# GSD: Complete Milestone

Archive project versions, create release documentation, and prepare for potential sequel projects.

## Workflow Source

This workflow is defined in detail at `.aiwcli/_gsd/workflows/complete-milestone.md`.

**CRITICAL:** Load the FULL content from `@.aiwcli/_gsd/workflows/complete-milestone.md`, READ its entire contents, and follow its directions exactly!

## Quick Reference

This command will:
1. Verify all phases complete and goals met
2. Generate MILESTONE-{version}.md summary
3. Create git tag for release
4. Archive project state to archive/v{version}/
5. Prepare for sequel project if continuing

## Usage

```
/gsd-complete-milestone
```

Run when all roadmap phases are complete and ready to ship.
