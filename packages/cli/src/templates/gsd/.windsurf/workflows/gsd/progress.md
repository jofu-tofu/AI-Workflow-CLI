# GSD: Progress

Display current project position, completed work, and recommended next steps.

## Workflow Source

This workflow is defined in detail at `.aiwcli/_gsd/workflows/progress.md`.

**CRITICAL:** Load the FULL content from `@.aiwcli/_gsd/workflows/progress.md`, READ its entire contents, and follow its directions exactly!

## Quick Reference

This command will:
1. Read current state from all project files
2. Calculate phase and task progress metrics
3. Show recent activity and commits
4. Identify active blockers
5. Recommend specific next action

## Usage

```
/gsd-progress
```

Quick command to see where you are and what to do next.
